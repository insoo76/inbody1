## 수정한 버그 (QA 중 발견)

1. **체중 이력 오인식** — `체 중`(OCR 분리) + 정상범위 `(52.0~70.4)` 를 이력으로 읽던 문제 → 라벨·괄호 필터링
2. **위상각 이력 공백** — 현재 위상각만 있을 때 `phase_angle_history` 비어 있던 문제 → 현재값 fallback
3. **510 PDF 이름 없음** — 회원번호 행에서 `회원 {번호}` fallback
4. **위상각 OCR 잡음** — 9.0↔3.0 튐 값 `_sanitize_phase_angles()` 로 필터
5. **체지방률 이력** — 체중(57~59)이 PBF로 섞이던 케이스 → anchor 기준 ±15%p 필터
