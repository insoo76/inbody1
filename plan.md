# SomaRx 서비스 고도화 기획서: 비주얼 대시보드 및 라이프스타일 처방 시스템

## 1. 기획 배경 및 목적
* **문제 정의:** 현재 SomaRx의 처방은 매우 전문적이고 상세하지만, 텍스트와 단순 바 차트 중심이라 사용자가 본인의 신체 상태 변화나 시급성을 한눈에 체감하기 어렵고 직관적인 인지가 부족함.
* **해결 방안:** 1. 수치 중심 데이터를 시각적 직관(인체 히트맵, 다차원 차트)으로 변환하여 **즉각적인 경각심과 성취감 제공**.
  2. 캐릭터 페르소나와 데일리 미션 트래커를 연동하여 **사용자의 지속적인 행동 변화 유도**.
* **기대 효과:** 서비스 리텐션(재방문율) 증가, 사용자 실질적 체성분 개선율 상승, 브랜드 로열티 강화.

---

## 2. 핵심 기능 1: 누적 트렌드 시각화 차트 (Progress Graph)

단순 수치 나열을 벗어나, 사용자의 '변화 방향성'을 직관적으로 보여주는 대시보드 영역입니다.

### 2.1. 데이터 요구사항
* **기준 데이터:** 최근 최대 5회차의 측정 데이터.
* **핵심 지표:** 체중(Weight), 골격근량(SMM), 체지방률(PBF).

### 2.2. UI/UX 구현 방안
* **날씨 아이콘 피드백:** 변화의 긍정/부정 상태를 아이콘으로 요약 표기.
  * ☀️ **맑음:** 골격근량 증가 & 체지방률 감소 (이상적인 근성장/다이어트)
  * ⛅️ **구름 조금:** 골격근량 유지 & 체지방률 유지 (유지어터)
  * ☔️ **비:** 골격근량 감소 & 체지방률 증가 (주의 필요)
* **미니 꺾은선그래프:** X축: 측정일 / Y축: 수치. 
  * 골격근량은 '파란색(성장)', 체지방률은 '주황색(경고)' 등 대비되는 색상을 사용하여 가독성 확보.
* **코멘트 자동화:** "지난달보다 골격근량이 **+1.5kg** 늘었어요! 훌륭한 성장입니다." 와 같은 동적 텍스트 제공.

## 3. 핵심 기능 2: 데일리 라이프스타일 미션 트래커 (Daily Tracker)

SomaRx의 '우선순위(높음/보통)' 처방 로직과 연동하여, 사용자별로 꼭 해야 하는 핵심 미션 3가지를 매일 체크하게 합니다.

### 3.1. 맞춤형 미션 추출 로직 (Dynamic Mission Generation)
인바디 분석 결과(지표 현황)에 따라 맞춤형 미션을 자동 할당합니다.
* **체수분 부족 시:** "오늘 하루 물 2.3L 이상 마시기 💧"
* **단백질 부족 시:** "매 끼니 손바닥 크기 단백질(계란/가슴살) 포함하기 🥩"
* **C형 (근력 부족형) 시:** "하체 중심 근력 운동 30분 하기 🏋️"
* **수면/성장점수 부족 시:** "밤 11시 이전에 취침하기 (7~8시간 수면) 😴"

### 3.2. UI/UX 구현 방안 (주간 뷰)
* **Weekly 체크리스트 포맷:**
  * 월요일~일요일까지 7일간의 달성 여부를 체크하는 UI.
  * 모바일 환경에서는 터치 시 가벼운 진동(Haptic FeedBack)과 함께 체크 아이콘 활성화.
* **달성률 보상 (Gamification):**
  * 주간 미션 달성률 80% 이상 시 '골드 뱃지' 부여 및 다음 인바디 측정 시 리워드 연동 가능성 제공.

## 4. 핵심 기능 3: 2D 인체 실루엣 히트맵 (Body Heatmap)

수치로만 채워진 부위별 분석을 한눈에 체감할 수 있도록 시각화한 그래픽 모델입니다.

### 4.1. 데이터 요구사항
* **입력 데이터:** 인바디 부위별 분석치 (오른팔, 왼팔, 오른다리, 왼다리, 몸통의 근육량/지방량 표준 비율).

### 4.2. UI/UX 구현 방안
* **아바타 실루엣 매핑:** * 사용자 성별에 따른 2D 신체 아바타 실루엣 중앙 배치.
* **컬러 오버레이 (히트맵):**
  * **체지방 과다 부위 (표준 이상):** 해당 신체 부위에 **빨간색(Warning Red) ~ 주황색 히트맵**을 투명하게 겹쳐서 표시.
  * **근육량 부족 부위 (표준 이하):** 해당 신체 부위에 **푸른색(Cold Blue) 히트맵**으로 표시.
  * **표준 부위:** 편안한 **연그린/그레이 컬러**로 안정감 부여.
* **효과:** 사용자가 텍스트를 읽기 전 "아, 내가 하체 근육이 파랗게 얼어있고(부족), 복부 주변이 붉구나(지방 과다)"를 0.5초 만에 인지할 수 있음.

## 5. 핵심 기능 4: 다차원 상태 분석 (방사형 차트 & 사분면 포지셔닝 맵)

지표 간의 불균형과 전체 인구군 내에서의 내 체성분 위치를 기하학적으로 보여줍니다.

### 5.1. 방사형 밸런스 차트 (Radar Chart)
* **목적:** 수분, 단백질, 무기질, 골격근량, 체지방량의 균형을 점검.
* **UI/UX 구현:** * 이상적인 정오각형(표준 균형) 가이드라인을 투명한 회색으로 깔아둔 뒤, 그 위에 사용자의 현재 상태를 진한 파란색 다각형으로 중첩 렌더링.
  * 특정 영양성분이 부족하여 정다각형 모양이 한쪽으로 찌그러질 경우 시각적 불균형을 즉시 체감 가능.

### 5.2. 상태 포지셔닝 사분면 맵 (Quadrant Map)
* **목적:** 골격근량과 체지방률 두 핵심 변수를 조합하여 직관적인 나의 현재 체형 포지션을 정의.
* **구조:**
  * **X축:** 골격근량 (낮음 → 높음)
  * **Y축:** 체지방률 (높음 → 낮음)
  * **4개 영역 정의:** 1사분면(비만형) / 2사분면(마른 비만형) / 3사분면(건강/슬림형) / 4사분면(근육형).
* **목표 가이드라인:** 현재 내 위치 좌표에 핀을 꽂고, 가야 할 이상적인 목표 지점(예: 3~4사분면 경계)까지 **점선 화살표(Path)**로 내비게이션 경로를 시각적으로 제시.

## 6. 핵심 기능 5: 계기판형 게이지 & 체형 페르소나 캐릭터

정량적 수치가 주는 딱딱함을 지우고 감성적 몰입과 시급성을 부여하는 시각적 장치입니다.

### 6.1. 종합 점수 속도계 게이지 (Speedometer UI)
* **대상 지표:** 인바디 종합 점수 및 성장 점수.
* **UI/UX 구현:** * 아날로그 속도계(Speedometer) 형태의 반원 게이지 사용.
  * 바늘이 현재 점수 영역(예: 82점)을 가리키며, 60점 이하(빨강/위험) → 70~80점(노랑/주의) → 80점 이상(초록/양호) 영역을 직관적인 컬러 스펙트럼으로 채움.

### 6.2. 체형 페르소나 캐릭터 (Persona Character)
* **목적:** 어려운 체성분 분류 기준을 나만의 '체형 캐릭터'로 매칭하여 친근감 형성.
* **캐릭터 정의 예시:**
  * **C형 (근력 부족형):** 귀여운 '자라나는 새싹 슬라임🌱' (말랑말랑하고 아직 성장이 필요한 캐릭터)
  * **I형 (표준 일자형):** '묵묵한 대나무 대리🐼' (표준이지만 단단함이 필요한 캐릭터)
  * **D형 (근육형 우수형):** '탄탄한 아기 호랑이🐯' (에너지가 넘치는 완성형 캐릭터)
* **게이미피케이션:** 다음 측정 시 체형이 개선되면 캐릭터가 진화(예: 슬라임에서 아기 호랑이로 변신)하는 애니메이션 효과 탑재.
